# Installation

There are three methods to install Paperback:

- [**App Store**](./appstore.md): The simplest and most stable way.
- [**TestFlight**](./testflight.md): Grants access to alpha/beta builds via Discord/Patreon.
- [**Sideloading**](./sideloading.md): Manual install for advanced users.

::: info Device Support (0.8)
The latest Paperback version (v0.8) is compatible with:

- iPhones running iOS 13.4 or later
- iPads running iPadOS 13.4 or later
- Macs running macOS 11.0 or later (Intel-based Macs are not supported)

The latest Paperback version (v0.9) is compatible with:

- iPhones running iOS 15.4 or later
- iPads running iPadOS 15.4 or later
- Macs running macOS 15.0 or later (Intel-based Macs are not supported)
  :::
