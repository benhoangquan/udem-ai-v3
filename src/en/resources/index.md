# Learning Resources

Curated resources to help you learn and master Machine Learning and Artificial Intelligence.

## Courses

### Online Courses

- **[Coursera - Machine Learning by Andrew Ng](https://www.coursera.org/learn/machine-learning)**
  - Comprehensive introduction to machine learning
  - Free to audit, certificate available

- **[Fast.ai - Practical Deep Learning](https://www.fast.ai/)**
  - Top-down approach to deep learning
  - Free and practical

- **[edX - MIT Introduction to Machine Learning](https://www.edx.org/course/introduction-to-machine-learning)**
  - University-level course
  - Free to audit

### University Courses

- **CS229 - Machine Learning (Stanford)**
  - [Course Materials](https://cs229.stanford.edu/)
  - Comprehensive notes and lectures

- **CS231n - Convolutional Neural Networks (Stanford)**
  - [Course Website](http://cs231n.stanford.edu/)
  - Deep dive into CNNs and computer vision

## Papers

### Foundational Papers

- **Attention Is All You Need** (Vaswani et al., 2017)
  - Introduced the Transformer architecture
  - [arXiv](https://arxiv.org/abs/1706.03762)

- **ImageNet Classification with Deep Convolutional Neural Networks** (Krizhevsky et al., 2012)
  - Breakthrough in image classification
  - [Paper](https://papers.nips.cc/paper/4824-imagenet-classification-with-deep-convolutional-neural-networks)

### Recent Papers

- **BERT: Pre-training of Deep Bidirectional Transformers** (Devlin et al., 2018)
  - [arXiv](https://arxiv.org/abs/1810.04805)

- **GPT-3: Language Models are Few-Shot Learners** (Brown et al., 2020)
  - [arXiv](https://arxiv.org/abs/2005.14165)
